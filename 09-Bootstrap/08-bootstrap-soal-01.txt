1. Apa itu CSS Framework?
Framework adalah pustaka kode yang mengabstraksi desain web umum dan membuat desain lebih mudah untuk diterapkan oleh 
pengembang di aplikasi web mereka. Secara sederhana, kerangka kerja CSS adalah kumpulan lembar gaya 
CSS yang disiapkan dan siap digunakan.
